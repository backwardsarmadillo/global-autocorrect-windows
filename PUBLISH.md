# Publish Checklist

Recommended public repo name:

```text
global-autocorrect-windows
```

Create an empty public repository on GitHub, then run:

```powershell
git remote add origin https://github.com/backwardsarmadillo/global-autocorrect-windows.git
git push -u origin main
```

After pushing, users can download `global-autocorrect-windows.zip` from the repo root.

For a nicer release page, create a GitHub release named `v0.1.0` and attach `global-autocorrect-windows.zip`.
